# GESTURE_CONTROL

![](ZZZ/ZZZ.png)

* Want to have some fun ?

* Want to control your computer using hand gestures ?

* We got you !

## COMPATIBILITY
* Tested on `ubuntu 22` and `bash`

* Open to resolve issues and contribute

## INSTALLATION
* `cd GESTURE_CONTROL`

* `chmod +x install.sh`

* `./install.sh`

## USAGE
* `python3 test.py`
